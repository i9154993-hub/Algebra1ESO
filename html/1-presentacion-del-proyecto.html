1. Presentación del proyecto

Descubriendo el lenguaje algebraico


 

¿Te imaginas poder traducir frases a símbolos matemáticos como si fueran códigos secretos?


En esta situación de aprendizaje descubrirás cómo utilizar el lenguaje algebraico y resolver ecuaciones para superar retos y resolver problemas reales. También utilizarás herramientas digitales para crear y presentar tus propias soluciones.

¿Qué vamos a aprender?

  1. Interpretar situaciones mediante lenguaje algebraico.
  2. Utilizar letras para representar números desconocidos.
  3. Resolver ecuaciones sencillas.
  4. Aplicar el álgebra a problemas cotidianos.

Nivel: 1.º ESO

Temporalización: 9 sesiones de 55 minutos

Materia: Matemáticas
